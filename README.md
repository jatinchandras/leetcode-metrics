# LeetCode Metrics

A simple web application to track and visualize your LeetCode problem-solving progress.

## Features
- Track solved problems
- Visualize progress with charts
- Simple, clean UI

## Getting Started

### Prerequisites
- Node.js (v14 or higher recommended)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/jatinchandras/leetcode-metrics.git
   ```
2. Navigate to the project folder:
   ```bash
   cd leetcode-metrics
   ```
3. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   ```

### Running the App
Start the development server:
```bash
npm start
# or
yarn start
```
Open [http://localhost:3000](http://localhost:3000) in your browser to view the app.
